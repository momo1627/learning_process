linux处理目录常用命令

man 命令 查看命令使用文档，使用参数和选项
  • 绝对路径：起点为根目录，/root/..
  • 相对路径，起点为当前目录，./为当前目录，../上级目录
  • ls:list directory
  • cd:change directory
  • pwd:show current directory
  • mkdir:make a new directory
  • rmkdir:remove an empty directory
  • cp 源文件 路径及目标文件:copy a file or directory
  • rm:remove a file or an empty directoy
  • mv 源文件 目标路径文件夹:move or change name
  • touch：修改文件或者目录的时间属性，或建立一个新的文件
  • whereis name: 搜索可执行文件，源文件和手册页面
  • which name: 在系统环境变量PATH中，搜索文件，打印可执行文件的完整路径
  • diff name1 name2: 比较两个文件差异
  • chmod filename: 更改文件读取，写入执行权限
文件内容查看
  • cat:从第一行显示内容
  • tac:从最后一行开始显示内容
  • nl：显示内容及行号
  • more：分页显示内容
  • less：分页显示内容且可向前翻页
  • head -n numer：查看前n行
  • tail -n number: 查看后n行

系统相关命令

  • export：显示所有环境变量
  • clear:清除命令行内容
备份压缩
  • gzip name:压缩文件，文件变为压缩格式
  • gunzip name：解压缩gz格式文件
  • gzcat name：查看gz文件，但不解压缩

shell 命令

  • echo $variable:输出变量
文件传输
  • lpr name:打印文件传至打印机
  • lpq:查看打印队列
  • lprm id:打印队列删除内容
